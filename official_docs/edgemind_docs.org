* LaTeX configuration

To use EdgeMind LateX classes, it is needed to add the EM LaTeX classes to the LaTeX path
(i.e. =/usr/share/texmf/tex/latex/= on Ubuntu). The easiest way to do this is to create a symbolic
link of the EM classes in the LateX path. 

Finally, run the =texhash= command as root to validate the changes.
